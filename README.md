# example-ndm

This example refers to the article on [nodejitsu.com](http://blog.nodejitsu.com/) and [npmawesome.com](http://npmawesome.com/posts/ndm) about `ndm`.

## Usage

    git clone https://github.com/npmawesome/example-ndm
    cd example-ndm
    npm install

    npm run install
    npm run start

    # then see ./logs

    # to cleanup
    npm run stop
    npm run remove
